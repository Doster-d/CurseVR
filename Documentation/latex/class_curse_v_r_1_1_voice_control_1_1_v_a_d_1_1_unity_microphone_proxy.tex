\doxysubsection{Подробное описание}
Provides an abstraction over Unity\textquotesingle{}s Microphone API for simplified access to audio input. 

This class encapsulates the initialization, management, and cleanup of Unity\textquotesingle{}s microphone system. It creates a continuous recording Audio\+Clip that can be accessed for real-\/time audio processing. The proxy pattern allows for easier testing and a more controlled interface to the Unity API.
