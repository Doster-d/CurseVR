\doxysubsection{Подробное описание}
A voice control system that integrates with Unity\textquotesingle{}s new Input System, using Voice Activity Detection (VAD) and a Web\+Socket-\/based ASR service.\hypertarget{README.md_autotoc_md1}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{README.md_autotoc_md1}

\begin{DoxyItemize}
\item Real-\/time voice activity detection
\item Web\+Socket-\/based communication with ASR service
\item Integration with Unity\textquotesingle{}s new Input System
\item Debug tools and visualization
\item Custom editor interface
\item Audio utilities for processing voice data
\end{DoxyItemize}\hypertarget{README.md_autotoc_md2}{}\doxysubsection{\texorpdfstring{Setup}{Setup}}\label{README.md_autotoc_md2}

\begin{DoxyEnumerate}
\item Install required dependencies\+:
\begin{DoxyItemize}
\item Voice Activity Detection (VAD) library
\item Native\+Web\+Socket
\item Uni\+Rx
\item Newtonsoft.\+Json
\end{DoxyItemize}
\item Configure the Input System\+:
\begin{DoxyItemize}
\item Create a new Input Action Asset if you haven\textquotesingle{}t already
\item Add a new Action Map named "{}\+Voice"{}
\item Add actions for each voice command you want to support
\end{DoxyItemize}
\item Set up the Voice Input Manager\+:
\begin{DoxyItemize}
\item Create an empty Game\+Object in your scene
\item Add the {\ttfamily Voice\+Input\+Manager} component
\item Assign your Input Action Asset
\item Configure the VAD Parameters
\item Set up the Voice Service Configuration
\item (Optional) Add an Audio Source for debug playback
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{README.md_autotoc_md3}{}\doxysubsection{\texorpdfstring{Configuration}{Configuration}}\label{README.md_autotoc_md3}
