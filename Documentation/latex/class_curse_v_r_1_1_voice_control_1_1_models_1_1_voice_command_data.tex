\doxysubsection{Подробное описание}
Data structure for voice commands received from the ASR service. 

This class represents a processed voice command after recognition by the ASR service. It contains the recognized command type, the original transcribed text, confidence level, and any parameters extracted from the command. This object is passed to subscribers of the On\+Command\+Recognized event in the voice command service.
