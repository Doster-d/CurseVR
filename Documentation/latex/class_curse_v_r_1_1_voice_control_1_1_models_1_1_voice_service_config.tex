\doxysubsection{Подробное описание}
Configuration settings for the voice command service. 

This serializable class contains all the connection and audio parameters needed to establish and maintain a connection with an Automatic Speech Recognition (ASR) Web\+Socket service. It can be configured in the Unity Inspector and saved as a Scriptable\+Object or directly in a Mono\+Behaviour component.

Gets or sets the size of the audio buffer in samples. 

Buffer size in samples (per channel)

This determines how much audio data is collected before sending to the ASR service. Larger buffers introduce more latency but may improve recognition accuracy. A value of 2048 at 16 k\+Hz represents about 128ms of audio.
